## Simulation of the standard iOS Contacts app to get some experience in SwiftData usage.

### Stack:
- Pattern: MVVM
- Navigation: NavigationStack, NavigationLink
- Data parsing: SwiftData, ObservableObject
- Views: List, ToolbarItem, TextField
